# VoiceControl 🎙️

> A BetterDiscord plugin that adds advanced voice channel controls (Mute, Unmute, Deafen, Disconnect) with hotkeys and UI panel.

---

## 📌 Features
- **Mute All**: Instantly mute everyone in your current voice channel.
- **Unmute All**: Remove mute from all members.
- **Deafen All**: Deafen all members (block hearing).
- **Disconnect All**: Kick everyone out of the voice channel.
- **Hotkeys Support**:
  - `Alt + M` → Mute All
  - `Alt + U` → Unmute All
  - `Alt + D` → Deafen All
  - `Alt + X` → Disconnect All
- **UI Panel**: Floating control panel with buttons for quick access.

---

## 🚀 Installation
1. Download the plugin file `VoiceControlPlusPlus.plugin.js`.
2. Move it into your BetterDiscord plugins folder:
   - Windows: `%appdata%/BetterDiscord/plugins`
   - macOS/Linux: `~/.config/BetterDiscord/plugins`
3. Restart Discord.
4. Enable the plugin from **User Settings → Plugins**.

---

## 🖥️ Usage
- Once loaded, you’ll see a small control panel at the bottom-right corner of Discord.
- Use the buttons or hotkeys to control the voice channel.
- Works only if you are connected to a voice channel.

---

## ⚠️ Notes
- Requires **BetterDiscord** installed.
- You must have the proper **permissions** (Mute/Deafen/Move Members) in the server to use these actions.
- This plugin is for **educational/admin use** only.
